### Local Development

1. Clone the repository

   ```bash
   git clone https://github.com/amajorai/landing.git
   cd landing
   ```

2. Install dependencies

   ```bash
   bun install
   ```

3. Set up environment variables

   ```bash
   cp .env.example .env.local
   ```

4. Start the development server

   ```bash
   bun dev
   ```
